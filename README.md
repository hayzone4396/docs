## 🚀 快速开始

### 环境准备

确保你的开发环境已安装：
- Node.js 18.0 或更高版本
- npm 或 yarn 包管理器

### 本地开发

```bash
# 克隆项目
git clone <repository-url>
cd vitepress-project

# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev
```

访问 http://localhost:5173 即可预览。

### 构建部署

```bash
# 构建生产版本
npm run docs:build

# 本地预览构建产物
npm run docs:preview
```

构建产物位于 `docs/.vitepress/dist` 目录。

## 🌐 部署方式

### 自动部署（推荐）
- **GitHub Pages** - 推送到 main 分支自动触发部署
- 通过 GitHub Actions 自动构建和发布

### 手动部署（备用）
- **Netlify** - 支持拖拽式部署
- 将 `docs/.vitepress/dist` 目录拖入 Netlify 即可

## 📝 内容创作

### 添加新文章

1. 在 `docs` 目录下创建 Markdown 文件
2. 在 `docs/.vitepress/config.js` 中添加导航和侧边栏配置
3. 使用 Markdown 编写内容

### Markdown 增强

支持以下 Markdown 扩展功能：
- 代码高亮
- 自定义容器
- 表格与列表
- 数学公式
- 图表绘制
